# Фото Деградер

GUI-приложение для намеренного ухудшения качества фотографий. Исходные файлы не изменяются — результат сохраняется отдельными JPEG-файлами в папку `degraded`.

## Онлайн-версия

Статическая браузерная версия находится в `index.html`. Она работает без сервера и без отправки фотографий в интернет: откройте файл напрямую или запустите локальный сервер:

```powershell
py -m http.server 8765
```

После этого откройте `http://localhost:8765`. Для публикации достаточно загрузить `index.html`, `styles.css` и `app.js` на GitHub Pages, Netlify, Cloudflare Pages или другой статический хостинг.

## Запуск

Дважды щёлкните **Запустить.bat** или выполните:

```powershell
py -m pip install -r requirements.txt
py metadata.py
```

## Возможности

- пакетная обработка отдельных фото и целых папок;
- JPEG-сжатие от 5 до 95%;
- уменьшение разрешения от 10 до 100%;
- дополнительное размытие;
- три готовых пресета;
- живое сравнение «до / после» прямо в главном окне при движении ползунков;
- увеличенное сравнение в отдельном окне;
- автоматическое удаление EXIF и других JPEG-метаданных;
- защита исходников и автоматические уникальные имена результатов.
